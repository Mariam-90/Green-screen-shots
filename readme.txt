



- Description:
This program is designed to remove the green screen background from an image and replace it with a different background image. It utilizes the OpenCV library for image processing and manipulation.

-Environment:
OpenCV, NumPy, Matplotlib, Sys 

- How To Run The Program: 
Open a terminal or command prompt
Run the program with the following command:
python program_name.py img_filename bk_filename ofilename
Replace program_name.py with the name of the Python script.
Replace img_filename, bk_filename, and ofilename with the filenames of the input image, background image, and output filename respectively.
Example:
python green_screen_remover.py input_image.jpg background_image.jpg output_image.jpg
